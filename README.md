A modern, high-performance personal portfolio platform designed to showcase professional work and experiences. Built with React and Vite, this project features smooth animations, interactive elements, and a clean, responsive UI/UX to create an engaging user experience.

## ✨ Key Features

- **Smooth Scrolling & Navigation:** Custom smooth scroll implementation for a cinematic browsing experience.
- **Dynamic Typing Animations:** Integrated `Typed.js` for engaging header introductions.
- **Interactive Data Counters:** Utilizing `react-countup` and `react-intersection-observer` for animated statistics that trigger when visible.
- **Modern Slider/Carousel:** Powered by `Swiper` for showcasing portfolio items and testimonials.
- **Sticky Header:** Enhanced navigation with `react-stickynode` for better accessibility.
- **Responsive Layout:** Fully optimized for all screen sizes, from mobile devices to large desktops.
- **Curated Sections:**
  - **About:** Personal introduction and professional summary.
  - **Experiences:** Timeline of professional journey and education.
  - **Services:** Overview of offered expertise and solutions.
  - **Portfolio:** Interactive gallery of completed projects.
  - **Testimonials:** Client feedback and social proof.
  - **Contact:** Integrated communication form.

## 🚀 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/issamovitch/my-portfolio
   ```
2. Navigate to the project directory:
   ```bash
   cd my-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Project

To start the development server:
```bash
npm run dev
```
The application will be available at the local URL provided in the terminal (usually `http://localhost:5173`).

## 📂 Project Structure

- `src/components`: Modular UI components (Header, Footer, Portfolio, About, etc.).
- `src/assets`: Images, fonts, and other static media assets.
- `public`: Static assets served directly.
- `src/style.css`: Global styles and custom CSS definitions.
